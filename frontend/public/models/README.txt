To get a realistic mannequin like in the reference image:

1. Download a realistic female mannequin model from:
   - Sketchfab: https://sketchfab.com/search?q=female+mannequin&type=models
   - Free3D: https://free3d.com/3d-models/mannequin

2. Make sure the file is in GLB or GLTF format

3. Rename it to: female-mannequin.glb

4. Place it in this folder (frontend/public/models/)

5. Refresh your browser

The mannequin will then look realistic with face, hair, and detailed features!
