## Step 4: Customer Interacts

### Action 1: Drag to Rotate
```
Customer drags mouse left/right
→ Model rotates 360°
→ Can see front, side, back views
```

**What they see:**
- Front view: Full garment display
- Side view: Profile and fit
- Back view: Back design
- Any angle: Complete visualization

### Action 2: Scroll to Zoom
```
Customer scrolls mouse wheel
→ Zooms in for details
→ Zooms out for full view
```

**What they see:**
- Zoomed in: Neckline details, fabric texture
- Zoomed out: Full body proportions
- Medium: Perfect viewing distance

### Action 3: Click Auto-Rotate
```
Customer clicks [▶ Rotate] button
→ Model automatically spins 360°
→ Hands-free viewing
```

**What they see:**
- Smooth rotation
- All angles automatically
- Can stop anytime
