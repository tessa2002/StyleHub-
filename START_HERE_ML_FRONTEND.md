## рҹҗӣ Troubleshooting

### "I don't see рҹӨ– AI/ML in sidebar"
вҶ’ Make sure you're logged in as **Admin** (not Customer/Staff/Tailor)

### "ML Dashboard shows error"
вҶ’ Check you're logged in (should have JWT token)
вҶ’ Open browser console (F12) to see error details

### "All models show 'Not Trained'"
вҶ’ Run: `cd backend && node ml/test-ml-models.js`
вҶ’ Then refresh the ML Dashboard page

### "Run Quick Test" doesn't work"
вҶ’ Models need to be trained first (see above)
вҶ’ Check backend is running on port 5000
